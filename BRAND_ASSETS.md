- `goldmine-sync-logo-wide-web.png` - Goldmine Sync nav logo
- `goldmine-bots-logo-original.png` - Goldmine Bots dynasty card logo
- `goldmine-edge-logo-wide-web.png` - Goldmine Edge dynasty card logo
- `logo-sync.svg` - SVG fallback/source asset
- `logo-bots.svg` - SVG fallback/source asset
- `logo-edge.svg` - SVG fallback/source asset

Before pushing hub updates, verify every `src="..."` asset referenced by `index.html` exists in this repository. GitHub Pages will show broken logos if the HTML references files that were not committed.
